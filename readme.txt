== Description ==

Allows site admins on a multi-site instance (WPMU) to edit users, only when that user belongs to the same site as the admin. Normally, this privilege is reserved on WPMU for network admins only.

== Installation ==

1. Download the plugin from your Plugins page in WordPress.
2. Activate the plugin.

== Frequently Asked Questions ==

= What purpose does this plugin serve? =

This plugin is useful for WordPress multi-site (WPMU) installations only. WPMU only permits network admins to edit users by default. This plugin additionally enables site admins to edit users, but only for users who belong to the same site as the admin.

= Can a site admin from site A edit users from site B? =

No.

= Can a network admin still edit users from sites A or B? =

Yes.
